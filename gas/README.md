# ProjectFlow (GAS版) セットアップ

Google スプレッドシートを DB、Google Drive を音声置き場とし、Google Apps Script の
ウェブアプリとして動作する ProjectFlow です。議事録の自動生成・タスク自動抽出・
ダッシュボード・経緯タイムライン・現状整理(ダイジェスト)を提供します。

## 1. スプレッドシートを作成

1. Google ドライブで新しいスプレッドシートを作成します(これがアプリの DB になります)。
2. メニューの **拡張機能 → Apps Script** を開きます。これがコンテナバインドされた
   Apps Script プロジェクトです。

## 2. ファイルを作成して貼り付け

Apps Script エディタで本ディレクトリの各ファイルを作成します。

- **`.gs` ファイル(スクリプト)**: `Main.gs` / `Ai.gs` / `Db.gs` / `Api.gs`
  - 左の「ファイル」横の **＋ → スクリプト** で作成し、同名(拡張子 `.gs` は自動)で
    内容をそのまま貼り付けます。
- **`.html` ファイル(HTML)**: `index.html` / `styles.html`
  - **＋ → HTML** で作成し、内容を貼り付けます。
- **`appsscript.json`(マニフェスト)**:
  - 左下の **プロジェクトの設定(歯車)** を開き、
    **「appsscript.json」マニフェスト ファイルをエディタで表示する** にチェックを入れます。
  - エディタ左に現れた `appsscript.json` を本リポジトリの内容で置き換えます。

## 3. スクリプト プロパティを設定

**プロジェクトの設定 → スクリプト プロパティ** で次を登録します。

| キー | 必須 | 用途 |
|---|---|---|
| `ANTHROPIC_API_KEY` | 必須 | 議事録・タスク抽出・現状整理(Claude API) |
| `OPENAI_API_KEY` | 任意 | 音声ファイルの文字起こし(Whisper API)。未設定でも文字起こし済みテキストの貼り付けで議事録は作成可能 |

## 4. 初回実行で権限を承認

エディタで適当な関数(例: `doGet` ではなく、まずは関数一覧から `ensureSheets_` など)を
一度実行するか、デプロイ後に初めてアクセスした際、Google の OAuth 同意画面が表示されます。
スプレッドシート・Drive・外部 HTTP(UrlFetchApp)へのスコープを承認してください。
初回アクセス時に 4 シート(projects / meetings / tasks / activities)が自動作成されます。

## 5. ウェブアプリとしてデプロイ

1. 右上の **デプロイ → 新しいデプロイ** を開きます。
2. 種類で **ウェブアプリ** を選択します。
3. 設定:
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: 自分のみ
4. **デプロイ** を押し、表示されたウェブアプリ URL を開きます。

コードを更新したら **デプロイ → デプロイを管理 → 編集 → 新バージョン** で再デプロイします。

## clasp を使う場合(任意)

ローカルから [clasp](https://github.com/google/clasp) でプッシュすることもできます。

```bash
npm i -g @google/clasp   # または npx clasp ...
clasp login
clasp clone <スクリプトID>   # またはコンテナの設定から取得
npx clasp push
```

`.gs` の読み込み順に依存はありません(GAS は全 `.gs` を1つのグローバルスコープに
連結します)。必要なら `.clasp.json` の `filePushOrder` で順序を固定できます。

## 制約

- **1 実行あたり約 6 分**の上限があります。このため文字起こし(`apiTranscribeAudio`)と
  議事録生成(`apiCreateMeeting`)は別 RPC に分割しています。
- **音声ファイルは 20MB 以下**にしてください(`google.script.run` のペイロード制限と
  Whisper の 25MB 制限のため)。UI 側で 20MB 超は送信前に中止します。
- **音声文字起こしには `OPENAI_API_KEY` が必須**です。未設定の場合は、外部で文字起こし
  したテキストを貼り付けて議事録を作成してください。
